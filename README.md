# h4d
## RUN LOCALLY
- `git clone REPO_URL`
- `cd h4d`
- `pip install pipenv`
- `pipenv install`
- `pipenv shell`

### run fake iPERMS app
- `python server.py`
### reset iPERMS app data
- `./reset_iPERMS.py`
### run autofill locally
NOTE: results will show up in `autofilled-documents` folder, each soldier will have their own folder
- `python autofill.py`
